About Stakeholders

List every person or group who affects or is affected by the project.

Interest = how much they care about the outcome.

Influence = power to impact the project.

High/Low  → Keep satisfied
Low/High  → Keep informed

Cost Hierarchy

Activity — cost of one task [e.g. design sprint, workshop]

Work Package — sum of related activities

Control Account — groups WPs for EVM tracking

Contingency — buffer for known risks (typically 10–15%)

BAC — Budget at Completion, the approved baseline

Management Reserve — buffer for unknown risks (typically 5%)

Budget = BAC + Mgmt Reserve
BAC = Estimate + Contingency
Gantt Chart
Task Name Start Date Days Priority Importance
About Gantt

Each bar = one task. Position = start date, width = duration.

Use working days for duration (5 = one week).

The critical path is the longest chain of dependent tasks — it sets the minimum project duration.

In PM² the Gantt is part of the Project Work Plan.
